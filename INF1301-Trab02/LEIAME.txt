Arquivo LEIAME
**************
Este documento fornece informações complementares para o TRAB02-JLR e sua instalação.
O objetivo do TRAB02 é o desenvolvimento de um módulo GRAFO. Este consiste em um grafo genérico bidirecionado de listas duplamente encadeadas. Dessa forma, a construção da estrutura grafo depende do módulo LISTA (genérica), que foi finalizado no TRAB01. 
O TRAB02 é baseado numa aplicação de Labirinto, mas que só deverá ser desenvolvida por completo no TRAB03.

3 - TESTES
*******************

Para realizar o teste das Listas:

Basta acessar a pasta "TRAB2-1" e clicar duas vezes no arquivo TesteListaGenerica.bat. Dessa forma, um arquivo chamado 
"logsaida.txt" será gerado com os resultados dos testes. Abra o arquivo logsaida.txt para analisar os resultados dos testes.

O arquivo TesteListaGenerica.bat possui o seguinte conteúdo: TRAB2-1.exe/sScripts/TesteListaGenerica.script >logsaida.txt


Para realizar o teste do Grafo de de listas:

Basta acessar a pasta "TRAB2-2" e clicar duas vezes no arquivo TesteGrafoGenerico.bat. Dessa forma, um arquivo chamado 
"logsaida.txt" será gerado com os resultados dos testes. Abra o arquivo logsaida.txt para analisar os resultados dos testes.

O arquivo TesteGrafoGenerico.script possui o seguinte conteúdo: TRAB2-2.exe/sScripts/TesteGrafoGenerico.script /logsaida.txt

Script -> Arquivo de instruções a ser interpretado pelo módulo de teste.
Log ->    Arquivo que conterá os resultados dos testes. Será criado, porém se já existir um arquivo log com o mesmo nome, ele será sobrescrito.
