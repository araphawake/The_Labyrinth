**********************
2 - SOBRE O PROGRAMA
**********************


	Este trabalho tem como objetivo intrumentar o módulo GRAFO para que seja garantida a análise de que ele funciona exatamente como o esperado, realizando o controle de cobertura - verifica se todos os pontos de relevência do código são visitados pelo menos uma vez. Apenas o TesteGrafoGenericoDEBUG.bat executa o modo DEBUG para cobertura e estrutura autoverificável. O TesteGrafoGenericoRELEASE.bat executa o modo Release com o teste de caixa fechada do módulo GRAFO - verifica que erros previstos não existem.


*******************
3 - TESTES
*******************

Para realizar o teste deste trabalho:

Acessar a pasta "TRAB4-1" e clicar duas vezes no arquivo TesteGrafoGenericoDEBUG.bat. Dessa forma, arquivos chamados 
"logSaidaTesteDebug.txt" e "logSaidaTesteVerticeNULLDebug.txt" serão gerados com os resultados dos testes de caixa fechada assim como os arquivos "logContagem.count" e "logContagemVerticeNULL.count" com os resultados de teste de caixa aberta. 
Abra o arquivo logSaidaTesteDebug.txt para analisar os resultados dos testes.
Abra o arquivo logSaidaTesteVerticeNULLDebug.txt para analisar os resultados dos testes.
Abra o arquivo logContagem.count para analisar os contadores do teste de caixa aberta do script TesteGrafoGenericoDEBUG.script.
Abra o arquivo logContagemVerticeNULL.count para analisar os contadores de teste de caixa aberta do script TesteVerticeNULL.txt.

Acessar a pasta "TRAB4-2" e clicar duas vezes no arquivo TesteGrafoGenericoRELEASE.bat. Dessa forma, um arquivo chamado "logSaida.txt"
será gerado com os resultados do teste de caixa fechada. 

Os arquivos finalizados com .count no nome podem ser abertos da seguinte forma:
1. Selecione o arquivo com o botão direito do mouse.
2. Em seguida, selecione a opção "Abrir Com".
3. Das opções listadas, selecione Bloco de Notas ou NotePad ou NotePad++. 


O arquivo TesteGrafoGenericoDEBUG.bat possui o seguinte conteúdo: 

TRAB4-1.EXE /sScript/TesteGrafoGenerico.script >logSaidaTesteDebug.txt
TRAB4-1.EXE /sScript/TesteVerticeNULL.script >logSaidaTesteVerticeNULLDebug.txt

O arquivo TesteGrafoGenericoRELEASE.bat possui o seguinte conteúdo: 

TRAB4-2.EXE /sScript/TesteGrafoGenerico.script >logSaida.txt
